# data_requirements_ecio2026.txt — ECIO 2026
#
# Structured manifest of the data files that the fetch (downloader) and process
# scripts in this directory rely on. make_app.py reads this file to decide
# whether a download can be skipped: when FORCE_DOWNLOAD is off, every file
# marked "required = yes" below must already exist in data/ for the download to
# be skipped; otherwise make_app runs the downloader.
#
# Format (simple, line-oriented; '#' starts a comment, blank lines ignored):
#   - Each file is a block introduced by a header line "[file: <pattern>]".
#   - <pattern> is a filename or a glob (e.g. ECIO26_*.pdf) resolved relative to
#     the data/ subdirectory of THIS conference folder.
#   - Inside a block, "key: value" lines give metadata. Recognized keys:
#       required     yes | no   (default yes) — whether the file must be present
#       produced_by  which script normally creates it (fetch / process / manual)
#       description  one-line human description
#       manual       how to obtain the file by hand if the downloader can't
#   - Continuation: a "manual" value may wrap onto following indented lines.
#
# manual_steps_written: 2026-05-27
# These manual-download instructions were accurate as of the date above; the
# conference website / planner layout can change, so re-verify if much later.

generated_for: ecio2026
manual_steps_written: 2026-05-27

[file: ECIO26_DetailedSchedule.pdf]
required: yes
produced_by: fetch
description: Detailed program schedule PDF — the wide A3 grid with every
  session, every time slot, and every talk title + speaker.
manual: Open https://www.ecio-conference.org/programme-26/ in a browser, click
  the "The detailed program can be downloaded here" link, and save the resulting
  PDF into this folder's data/ subdirectory as exactly ECIO26_DetailedSchedule.pdf .
  (The current direct URL is
  https://www.ecio-conference.org/wp-content/uploads/2026/05/ECIO26_DetailedProgramSchedule_21_5.pdf
  but the date suffix changes when the organisers re-issue the schedule, so
  always re-derive the link from the programme page rather than hard-coding it.)

[file: 2026_ecio_agenda_of_sessions.pdf]
required: no
produced_by: fetch
description: Agenda-of-Sessions PDF — a clean one-table-per-day overview that,
  unlike the detailed schedule, prints the LOCATION of every non-talk event
  (the registration desk, the coffee/lunch foyers, the Welcome Reception and
  Gala venues, the student-event rooms) and the room column each session block
  sits under. The processor uses it to fill the location of any session the
  detailed schedule left without one, and to add the daily Registration /
  Coffee Break / Lunch rows the wide grid omits. Optional: missing it just
  reverts session locations to detailed-schedule rooms and drops those
  logistics rows.
manual: This PDF is NOT linked from the CMS programme page; it is served
  from the publisher's media CDN. The current direct URL is
  https://opticaorg-dev-cac7d2csctagc8bm.z01.azurefd.net/$web/optica/media/files/events/ecio/2026/2026_ecio_agenda_of_sessions.pdf
  — download it and save into this folder's data/ subdirectory as exactly
  2026_ecio_agenda_of_sessions.pdf . If that CDN path has rotated, look for an
  "Agenda of Sessions" link on the official event page
  (https://www.optica.org/events/topical_meetings/european_conference_on_integrated_optics_(ecio)/)
  and save whatever it points to under the same filename.

[file: ECIO26_InvitedSpeakers.html]
required: yes
produced_by: fetch
description: Invited-speakers page HTML — lists every invited speaker as a
  (Name, Affiliation, Talk Title) triple grouped under SC1..SC7 sub-committee
  headers. The detailed-schedule PDF prints only speaker names in each talk
  cell, with no affiliation; the processor parses this HTML to fill the
  matching talks' institutions in the final JSON. Missing this file does not
  fail the processor, but the resulting JSON loses all invited-speaker
  affiliations, so the build pipeline treats it as required.
manual: Open https://www.ecio-conference.org/invited-speakers-2/ in a browser.
  Use "Save Page As…" (Ctrl/Cmd-S) and choose the "Webpage, HTML Only" option
  (NOT "Webpage, Complete"), saving into this folder's data/ subdirectory as
  exactly ECIO26_InvitedSpeakers.html . The URL is stable across re-issues —
  unlike the PDFs above it carries no date suffix — so no link re-derivation
  is needed.

# The three event-site schedule pages below are the richest enrichment source:
# the full program is mirrored on the official event site as a per-day schedule
# whose "Detailed View" cells carry the COMPLETE author list (with each
# author's affiliation) AND the abstract for every talk — neither of which the
# detailed-schedule PDF renders. The processor cross-references them by session
# code + talk title to fill author lists and abstracts on the PDF-derived oral
# talks, and to populate the poster sessions the PDF leaves empty (the wide
# grid lists no individual posters). All three are optional: missing them just
# reverts oral talks to PDF-only authorship and leaves the poster sessions
# empty. They are browser-rendered (a JavaScript single-page app behind bot
# protection), so the fetcher drives a headed Chromium via Playwright rather
# than a plain HTTP download; the manual fallback below reproduces those steps
# by hand.

# The conference planner is the only public source that lists each technical
# session's PRESIDER(s). The fetcher drives a headless Chromium, expands every
# day/session/"See More…" control (as in the sibling fetchers), and saves the
# fully-expanded DOM. The processor keys the presider
# map by session code (M1B, T2A, W3B, …) — the same code the PDF uses — and
# attaches presider + affiliation to the matching session. Optional: missing
# this file just leaves every session without a presider. (We deliberately do
# NOT download the planner's Program+Abstracts PDF/CSV — the PDFs above are the
# authoritative program; the planner is used only for presiders.)

[file: ECIO26_planner_expanded.html]
required: no
produced_by: fetch
description: Fully-expanded conference-planner DOM (outerHTML). Source of the
  per-session presider names + affiliations.
manual: Open https://ecio2026.abstractcentral.com/planner.jsp in a browser,
  wait for the program to render, then expand EVERY day row, every session row

  ECIO26_planner_expanded.html , prefixed with a "<!DOCTYPE html>" line.
